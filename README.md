# FFSwap
Fork of hsse's ffswap to work in most systems.  Works and tested with kitty


## Installation
To install, run

```bash
make
```

## Config
Images need to be placed in ~/.config/fastfetch
Images + colors neet to be manually configured in [main](./ffswap/__main__.py) for now
